在Mac下搭建自己的blog

2017-06-20

安装Xcode
app store

安装node.js
官网

注册git

安装hexo
npm install -g hexo-cli –no-optional

创建文件夹
mkdir blog
cd blog
hexo init

生成静态网页
hexo generate / 生成一套静态网页 /
hexo server / 在服务器上运行 /
可在http://localhost:4000查看

撰写blog
hexo new post “first blog”

上传至Github
安装git部署插件
npm install hexo-deployer-git –save

配置_config.yml 文件

deploy:
type: git
repo: https://github.com/用户名/用户名.github.io
branch: master

同步
hexo clean
hexo generate
hexo deploy
